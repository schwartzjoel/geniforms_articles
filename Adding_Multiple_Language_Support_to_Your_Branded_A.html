Adding Multiple Language Support to Your Branded App

Translating Static App Text

We support using multiple languages in your branded app through a simple dictionary-based approach. By default, we have support for English, Spanish, and Dutch languages; however, our platform can support any number of languages using a language file.


Navigate to the Branding Center > App Settings page.

You can download an Excel file containing all static text in the app and the default English translation.

Edit this Excel file to change or translate the text as desired.



If you want to support multiple languages, add columns next to the default English and give these columns a header/title value of the language name.


Once you have all the desired languages entered into the Excel file, then upload this translation file back into the "App Setting" page.


This will make the translations available to all users of your custom-branded app.


Suppose you have uploaded multiple languages in the translations file. In that case, the app will automatically present a language selection dropdown on the app login and Settings screens.



The user can choose their desired language and change it anytime.


As we add new features to the app, the translation file will increase in size with new static text values.

So you will need to keep an eye on the "App Settings" page to ensure you add translations for new words/phrases added to the app.
