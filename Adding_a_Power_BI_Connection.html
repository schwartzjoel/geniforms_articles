Adding a Power BI Connection


To add a Power BI Connection, navigate the Side Menu>Connected Data>Connections.


More information on how to add a Connection can be found here, Adding Data Connections.




On the Connections page:

  • Click the ADD CONNECTION button on the top right to bring up a list of connections
  • Click on Power BI option


Next, you'll need your:

  • Power BI Application ID
    This identifies your Power BI application to Azure Active Directory.
  • Application Secret
    This secures your Power BI application with Azure Active Directory.
  • Workspace ID (optional)
    Add an ID for your custom PowerBI workspace. Leave blank to use the default My Workspace.
    Note, can be obtained via URL in browser when viewing workspace.


This can be obtained by following this guide:
https://docs.microsoft.com/en-us/power-bi/developer/register-app#register-with-the-power-bi-application-registration-tool



When registering and required to enter a value for Home Page URL, copy/paste the Redirect URL (omitting "/oauth2/powerbicallback") provided on our Connections Page, Power BI Connection. 
After succesfully registering your application, copy/paste the Application ID and Secret into our Connections Page, Power BI Connection, where required.



Once your Power BI connection details are added, click Save in the top right corner.


If your Application ID and Application Secret keys are correct, an Authorise button will appear, allowing you to authenticate the connection.



By this point, your Power BI connection has been configured at an organizational level, and you can start adding Power BI Connectors to forms and linking them to your Power BI datasets.
