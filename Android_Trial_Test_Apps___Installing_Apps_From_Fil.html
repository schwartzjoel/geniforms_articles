Android Trial/Test Apps - Installing Apps From File

We generally provide trial and test versions of your branded app via a direct download of the Android APK file.


To install on this your Android devices:


1. Download the attached zip file and unzip on your computer. You will then have an APK file.


2. Copy the APK file to your Android device.  

This is usually done using a USB cable and whatever software your particular device model came with. It's the same process as you would use to copy music or pictures.

Make a note of the device folder you copy the APK file to.


3. On your Android device, use a file explorer app such as ES File Explorer to browse your device's files.

Using your file explorer app, go to the folder containing your APK and tap it to install.


NOTE: Some Android devices don't allow the installation of apps from what Google refers to as "unknown sources."
Fortunately, it's usually very easy to enable this functionality.

Simply pop over to your phone's Settings menu (usually found by hitting the menu button from the home screen) and tap the option for Applications.

You should see an option that says "Unknown sources."
Fill the check box next to it, then click OK on the resulting popup alert.


4. Once the app is installed, you can log in using any existing Tap-to or Your Company account that you may have.
