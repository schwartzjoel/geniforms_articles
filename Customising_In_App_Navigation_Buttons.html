Customising In-App Navigation Buttons

Configure up to 4 navigational buttons to control in-app behavior on all screen types (Data Entry, Listing, Details, Mapping, and Board of Icons).


When viewing a screen's design, click the title/navigation bar to display its properties on the righthand side of the screen, and under Advanced Options are configurable Action Buttons.


Data Entry Form Designer


Each Action Button's (Left1, Left2, Right1, Right2) icon, text, styling, and user interaction can be configured as desired. Click a button to display its properties and configure.



The User Interaction defines the action when the button is tapped in-app. Button Visibility allows for a formula that returns a true or false result to control the button's visibility. 


Also, the Button's Text property supports formula building (Hammer icon), allowing for dynamic text driven from formula or form field values.
