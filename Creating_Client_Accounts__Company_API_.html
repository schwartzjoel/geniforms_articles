Creating Client Accounts (Company API)

Using our Company API endpoints, you can tightly integrate our platform with your own.


For instance, you could add a sign-up page to your own marketing website. This would allow potential clients to sign-up for trials under your master vendor account directly from your site!


An example use of this can be seen here:


This is made possible by incorporating our API Explorer's Company POST feature and more into your own website, for use by IT personal within your organization.

  • API Explorer
  • Company POST


The API is exposed using web services in a REST approach, providing a flexible and simple interface that can be accessed from just about any programming language.


The APIs support both JSON and raw XML formats.

NOTE: XML requests must have each tag element in alphabetical order.


A simple example of the data, in JSON format, to POST a generic company named 'A Company Name' is shown here:

{
  "Name": "A Company Name",
  "Description": "General",
  "Country": "United States",
  "TimeZone": "Pacific Standard Time",
  "DateFormat": "yyyy-MM-dd",
  "TimeFormat": "hh:mm:ss",
  "FillRepeatRows": false,
  "RerouteResetEmails": false,
  "DefaultSenderEmail": "email@company.com",
  "DeviceLimit": 0,
  "PremiumUserLimit": 0,
  "StandardUserLimit": 0,
  "LiteUserLimit": 0,
  "UserLimit": 0,
  "AddSupportUser": false,
  "CopyCustomIcons": false,
  "IntegrationKey": "",
  "VendorId": 
}

If you're still uncertain which node your account resides on, simply look at the URL in your browser:

  • secure = Americas
  • secure-eu = Europe/Africa/Middle East
  • secure-au = Oceania/Asia
